Variance

變異數(方差)

$$\sigma ^{2}=\frac{\sum (x-\bar{x})^{2}}{n}$$

x = Observations given $\bar{x}$ = Mean
n = Total number of observations

Standard Deviation

標準差(標準偏差)

$$S = \sigma = \sqrt{\frac{\sum (x-\bar{x})^{2}}{n}}$$

x = Observations given $\bar{x}$ = Mean
n = Total number of observations

coefficient of variation

變異係數

$$c_v = {\sigma \over \mu }$$

是機率分佈離散程度的一個歸一化量度
其定義為標準差$\sigma$與平均值$\mu$之比
變異係數只對由比率純量計算出來的數值有意義。

interquartile range, IQR

四分位距

四分位差（Quartile Deviation, QD），
是$Q_{1},Q_{3}$的差距的一半，
即$\mathrm{QD} = \frac{Q_3 - Q_1}{2}$

是描述統計學中的一種方法，
以確定第三個四分位數和第一個四分位數的差值
（即$Q_{1},Q_{3}$的差距）
