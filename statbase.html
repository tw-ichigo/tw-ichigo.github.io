Mean

平均數

$$\bar{x}=\frac{\sum x}{n}$$

x = Observations given
n = Total number of observations

Median

中位數

If n is odd, then M = $(\frac{n+1}{2}^{th})$ term
If n is even, then M = $\frac{(\frac{n}{2})^{th}term+(\frac{n}{2}+1)^{th}term}{2}$

n = Total number of observations

Mode

眾數

The value which occurs most frequently

指一組數據中出現次數最多的數據值。
眾數可能是一個數，但也可能是多個數。

Variance

變異數(方差)

$$\sigma ^{2}=\frac{\sum (x-\bar{x})^{2}}{n}$$

x = Observations given $\bar{x}$ = Mean
n = Total number of observations

Standard Deviation

標準差(標準偏差)

$$S = \sigma = \sqrt{\frac{\sum (x-\bar{x})^{2}}{n}}$$

x = Observations given $\bar{x}$ = Mean
n = Total number of observations

Population and Sample

母體和樣本

Population: $$\sigma ^{2}=\frac{\sum (x-\bar{x})^{2}}{n}$$ Sample: $$s^{2}=\frac{\sum (x-\bar{x})^{2}}{n-1}$$

x = Observations given $\bar{x}$ = Mean
n = Total number of observations
