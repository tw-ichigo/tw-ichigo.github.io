集中趨勢(central tendency）或中央趨勢

Arithmetic mean

算數平均數

$$\tfrac{x_1 + x_2 + x_3 \ldots + x_n}{n}$$

稱為平均數
機率分佈的期望值之不偏估計。

Median

中位數

If n is odd, then M = $(\frac{n+1}{2}^{th})$ term
If n is even, then M = $\frac{(\frac{n}{2})^{th}term+(\frac{n}{2}+1)^{th}term}{2}$

n = Total number of observations
將所有觀測值按大小排序後在順序上居中的數值。

Mode

眾數

The value which occurs most frequently

指一組數據中出現次數最多的數據值。
眾數可能是一個數，但也可能是多個數。

Geometric mean

幾何平均數

$$(x_1 \times x_2 \times x_3 \ldots \times x_n)^{\frac{1}{n}}$$

觀測值的乘積之觀測值個數方根

Harmonic mean

調和平均數

$$\frac{n}{\frac{1}{x_1} + \frac{1}{x_2} + ... + \frac{1}{x_n}}$$

觀測值個數除以觀測值倒數的總和

weighted mean

加權平均數

$$\bar{x} = \frac{ \sum_{i=1}^n w_i x_i}{\sum_{i=1}^n w_i}$$

考慮不同群資料貢獻程度不同時的算數平均數

truncated mean

截尾平均數

例如，四分平均數(Interquartile_mean)
正是忽略25%前及75%後的資料後所得的算數平均數。

忽略特定比例或特定數值之外的極端值後所得的平均數。

midrange

中程數

$$\frac{\min (x) + \max (x)}{2}$$

最大值與最小值的算數平均數，
又稱全距中值

midhinge

中樞紐

$$\frac{Q_1 + Q_3}{2}$$

第一[四分位數]與第三[四分位數]的算數平均數

trimean

三均值

$$\frac{Q_1 + 2Q_2 + Q_3}{4}$$

考慮三個四分位數的加權平均數

Winsorized_mean

極端值調整平均數

舉例來說，
考慮10個觀測值（由小到大排列為$x_1$至$x_{10}$）的情況下，
10%的極端值調整平均數為 :: $\frac{\overbrace{x_2 + x_2} + x_3 + x_4 + x_5 + x_6 + x_7 + x_8 + \overbrace{x_9 + x_9}}{10}$，
: 其中分別以$x_2$和$x_9$取代了$x_1$和$x_{10}$。

以最接近的觀測值取代特定比例的極端值後取得的算數平均數。

Quadratic mean(Root Mean Square)

方平均數(方均根,RMS)

$$M = \sqrt{\sum_{i=1}^n x_i^2 \over n} = \sqrt{x_1^2 + x_2^2 + \cdots + x_n^2 \over n}$$ $$連續函數: f_{\mathrm{rms}} = \sqrt {{1 \over {b-a}} {\int_{a}^{b} {[f(x)]}^2\, dx}}$$

方均根值並非所有模型均適用， 只有在數值分布呈現常態分布時才適用。

arithmetic–geometric mean (AGM)

算術-幾何平均數

$u_0=a,v_0=b$ $$u_{n+1} = \sqrt{u_nv_n}$$ $$v_{n+1} =\frac{u_n + v_n}2$$

這兩個數列收斂於相同的數，這個數稱為x和y的算術-幾何平均數，記為M(x, y)

Geometric–harmonic mean

幾何-調和平均數

$$g_1 = \sqrt{xy}.$$ $$h_{1} = \frac {{2xy}} {{x + y}}$$ $$g_{n+1} = \sqrt{g_n h_n}.$$ $$h_{n+1} = \frac{2}{\frac{1}{g_n} + \frac{1}{h_n}}$$

這兩個數列收斂於相同的數，這個數稱為x和y的幾何-調和平均數，記為ghm(x, y)。

Heronian mean

希羅平均數

數學上，希羅平均數是指兩個非負的實數的一種平均，它的公式為 $$H = \frac{1}{3} \left(A + \sqrt{A B} +B \right).$$ 希羅平均數也可以看成算術平均數與幾何平均數的加權平均數。 $$H = \frac{2}{3}\cdot\frac{A+B}{2} + \frac{1}{3}\cdot\sqrt{A B}.$$

圓台的體積,B1是一個底面的面積, B2 是另一個底面的面積, h為高。 $$V = \frac{h}{3}(B_1+B_2+\sqrt{B_1 B_2})$$

Mean inequality

平均數不等式

如果$x_1, x_2, \ldots, x_n$是正數，則 $$H_n \le G_n \le A_n \le Q_n$$

其中： $$H_n = \dfrac{n}{\displaystyle \sum_{i=1}^{n} \dfrac{1}{x_i}} = \dfrac{n}{\dfrac{1}{x_1}+\dfrac{1}{x_2}+\cdots+\dfrac{1}{x_n}}$$ $$G_n=\sqrt[n]{\prod_{i=1}^nx_i}=\sqrt[n]{x_1x_2\cdots x_n}$$ $$A_n = \dfrac{\displaystyle \sum_{i=1}^{n} x_i }{n} = \dfrac{x_1+x_2+\cdots+x_n}{n}$$ $$Q_n = \sqrt{\dfrac{\displaystyle \sum_{i=1}^{n} x_i^2}{n}} = \sqrt{\dfrac{x^2_1+x^2_2+\cdots+x^2_n}{n}}$$ 若且唯若$x_{1}=x_{2}=\cdots =x_{n}$ ，等號成立。
即對這些正數：調和平均數 ≤ 幾何平均數 ≤ 算術平均數 ≤ 平方平均數（方均根）
