集中趨勢(central tendency）或中央趨勢

Arithmetic mean

算數平均數

$$\tfrac{x_1 + x_2 + x_3 \ldots + x_n}{n}$$

稱為平均數
機率分佈的期望值之不偏估計。

Median

中位數

If n is odd, then M = $(\frac{n+1}{2}^{th})$ term
If n is even, then M = $\frac{(\frac{n}{2})^{th}term+(\frac{n}{2}+1)^{th}term}{2}$

n = Total number of observations
將所有觀測值按大小排序後在順序上居中的數值。

Mode

眾數

The value which occurs most frequently

指一組數據中出現次數最多的數據值。
眾數可能是一個數，但也可能是多個數。

Geometric mean

幾何平均數

$$(x_1 \times x_2 \times x_3 \ldots \times x_n)^{\frac{1}{n}}$$

觀測值的乘積之觀測值個數方根

Harmonic mean

調和平均數

$$\frac{n}{\frac{1}{x_1} + \frac{1}{x_2} + ... + \frac{1}{x_n}}$$

觀測值個數除以觀測值倒數的總和

weighted mean

加權平均數

考慮不同群資料貢獻程度不同時的算數平均數

truncated mean

截尾平均數

例如，四分平均數(Interquartile_mean)
正是忽略25%前及75%後的資料後所得的算數平均數。

忽略特定比例或特定數值之外的極端值後所得的平均數。

midrange

中程數

$$\frac{\min (x) + \max (x)}{2}$$

最大值與最小值的算數平均數，
又稱全距中值

midhinge

中樞紐

$$\frac{Q_1 + Q_3}{2}$$

第一[四分位數]與第三[四分位數]的算數平均數

trimean

三均值

$$\frac{Q_1 + 2Q_2 + Q_3}{4}$$

考慮三個四分位數的加權平均數

Winsorized_mean

極端值調整平均數

舉例來說，
考慮10個觀測值（由小到大排列為$x_1$至$x_{10}$）的情況下，
10%的極端值調整平均數為 :: $\frac{\overbrace{x_2 + x_2} + x_3 + x_4 + x_5 + x_6 + x_7 + x_8 + \overbrace{x_9 + x_9}}{10}$，
: 其中分別以$x_2$和$x_9$取代了$x_1$和$x_{10}$。

以最接近的觀測值取代特定比例的極端值後取得的算數平均數。
